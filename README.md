# django-simple-bank-management-system
A simple Bank Management System built with Django. It demonstrates CRUD operations, account management, deposits, withdrawals, and Django ORM for beginners.

# 🏦 Django Bank Management System

A beginner-friendly banking management system developed using **Python** and **Django**.

This project was built to practice Django fundamentals such as Models, Views, Templates, URLs, Forms, ORM, and CRUD operations.

---

## Features

- ✅ Create New Account
- ✅ View All Accounts
- ✅ Search Account
- ✅ Deposit Money
- ✅ Withdraw Money
- ✅ Update Account Information
- ✅ Delete Account
- ✅ Django Admin Panel
- ✅ SQLite Database
